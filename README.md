# Student Registration System (PHP & MySQL)

This project is a simple student registration system developed using PHP and MySQL.

## Features
- Student registration form
- Stores data in MySQL database
- Uses phpMyAdmin for database management

## Technologies Used
- PHP
- MySQL
- HTML
- XAMPP

## How to Run
1. Install XAMPP
2. Copy project folder to htdocs
3. Start Apache and MySQL
4. Open browser and run:
   http://localhost/student/
